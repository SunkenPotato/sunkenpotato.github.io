What do the colors mean?

The colors next to the project names stand for the current state of the project.

Colour Meaning
● This project is currently finished or at a stable version.
● This project is currently under development and / or not accessible to the public yet.
● This project has been abandoned.
● I'm taking a break from working on this project / or something else.
